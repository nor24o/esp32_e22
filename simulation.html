Packet Structure (38 bytes total)

Header (6 bytes):
 [0-1] magic_word — network ID (0xE22A)
 [2] source_id — sender node ID
 [3] target_id — destination node ID
 [4] msg_type — message type (1-8)
 [5] msg_id — rolling counter 0-255
Payload (32 bytes, union by type)

Message Types

1 = TELEMETRY — periodic state broadcast
2 = COMMAND — pump ON/OFF, mode change
4 = CONFIG_GET — request config values
5 = CONFIG_SET — write config to flash
6 = CONFIG_RESP — config reply
7 = STATS_GET — request runtime stats
8 = STATS_RESP — stats reply

Node IDs

0x01 = Gateway Controller
0x02 = Tank Controller (this device)
0x03 = Pond Controller
0xFF = BROADCAST (all nodes receive)

system_flags Bits (telemetry)

bit 0 = auto_mode  bit 1 = P1  bit 2 = P2
bit 3 = P3         bit 4 = pond pump
